# 🌙 99 Names App

A Duolingo-inspired, Islamic learning application for mastering the 99 Beautiful Names of Allah (Asma'ul Husna).

## Vision

Create a beautiful, engaging and spiritually focused learning experience that combines:

- 📖 Authentic learning
- 🎮 Gamification
- 📱 Mobile-first design
- 🌐 Progressive Web App (PWA)
- 🧠 Spaced repetition
- 📊 Progress tracking

## Current Status

- ✅ Premium single-file prototype
- ✅ Flashcard lesson
- ✅ Match-the-pairs activity
- ✅ Word builder activity
- ✅ XP, Hearts, Progress and Streak system
- ✅ Local storage support
- 🚧 Modular architecture (in progress)
- 🚧 Complete 99-name database
- 🚧 Offline PWA support
